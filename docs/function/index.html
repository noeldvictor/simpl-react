    public simpl(options: Object): * source 

    import {simpl} from 'simpl-react/lib/decorators/simpl'

    Decorator to wrap your main app

    Params:

    NameTypeAttributeDescription
    options Object

    An object of options.
    options.authid string

    The user id for authenticating on the WAMP Router. This will the user's id on Simpl-Games-Api
    options.password string

    The password for authenticating on the WAMP Router.
    options.url string

    The URL of the WAMP router.
    options.progressComponent function
    • optional

    A customized Component to show the App's connection state. The component will receive a connectionStatus prop which can have one of the following values:

    • connecting: The initial value. The app is not connected to the WAMP Router, yet.
    • connected: The app is connected and authenticated, but it still needs to download data.
    • loaded: The app has downloaded all the relevant data.
    • offline: The connection was dropped.
    options.topics function

    A function returning a list of topics to subscribe to.
    options.root_topic string

    The root topic for your model. This will be used for the 'model' prefix.
    options.prefixes Object
    • optional

    An object mapping names to topic prefixes, to be used as shortcuts.
    options.loadAllScenarios boolean

    If false, load only world scenarios and this user's Scenarios. If true, load all Scenarios for the subscribed runs.

    Return:

    *

    Example:

    export default simpl({
  authid: 123,
  password: 'password',
  url: 'ws://example.com/ws',
  progressComponent: MyProgressComponent,
  topics: () => ['topic1', 'topic2'],
  root_topic: 'org.example.namespace',
  prefixes: {
    special: 'org.example.namespace.special.shortcut'
  },
  loadAllScenarios: false
})(MyComponent);
