    public CurrencyInput: ReactElement source 

    import {CurrencyInput} from 'simpl/lib/components/forms/CurrencyInput.react'

    A component that allows entering a decimal value and always shows it with fixed precision, prefixed by a currency symbol.

    Default validation options:

    • errors: ['isCurrency', min, max],
    • sanitizers: ['toFloat']
    • formatters: [decimalPlaces]

    Default props: {type: 'number', decimalPlaces: 2, currency: '$'}
