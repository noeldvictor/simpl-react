    public CONNECTION_STATUS: {"CONNECTING": string, "CONNECTED": string, "LOADED": string, "OFFLINE": string} source 

    import {CONNECTION_STATUS} from 'simpl-react/src/constants.js'

    A constant describing all the possible connection status.

    Properties:

    NameTypeAttributeDescription
    CONNECTING string
    • default: 'connecting'

    The initial value. The app is not connected to the WAMP Router, yet.
    CONNECTED string
    • default: 'connected'

    The app is connected and authenticated, but it still needs to download data.
    LOADED string
    • default: 'loaded'

    The app has downloaded all the relevant data.
    OFFLINE string
    • default: 'offline'

    The connection was dropped.
